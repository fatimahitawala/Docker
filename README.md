# Docker
Understanding of Docker Architecture
### WHAT IS Docker
Docker is a Containerized tool, to automate the application deployment for them to run in any environment or any OS effeciently. It packages all its dependencies, libraries along with the code.
In the architecture there is docker-client, docker host, and docker registry. Docker client uses rest API methods to talk to docker deamon. Docker deamon do heavy lifting of building, running and distributing of your containers.
